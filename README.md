# Crack Story Builder Tool

## v1.0
  
- 크랙 스토리 빌더의 스토리 편집 페이지에서 이미지(또는 키워드북)를 대량으로 관리할 때 발생하는 번거로운 반복 작업을 한 번의 클릭으로 해결해 주는 템퍼몽키 스크립트입니다.

- 주요 기능:

❌ 이미지 전체 삭제: 업로드된 모든 이미지를 자동으로 하나씩 삭제하고 확인 팝업까지 클릭합니다. (키워드북 페이지에서 사용시 키워드북 전체 삭제에도 사용 가능) 

🔢 순서 번호 자동 채우기: 연필 버튼을 순서대로 클릭하여 1번부터 차례대로 숫자를 입력하고 저장합니다.

↕️ 전체 창 축소/확대: 모든 이미지 또는 키워드북 상세 창을 한꺼번에 접거나 펼쳐서 가독성을 높입니다.

📝 빈 설명칸 '1' 채우기: 비어있는 이미지 설명(프롬프트) 칸에만 자동으로 '1'을 입력하여 유효성 검사를 통과하도록 돕습니다.

⚙️ 설치 방법:
https://github.com/candid0649/Crack_storybuilder_tool/blob/main/storybuilder.user.js

⚠️ 주의사항: 모든 작업이 완료된 후, 페이지 내의 [저장하기] 또는 [스토리 저장] 버튼을 반드시 눌러야 서버에 최종 반영됩니다.
